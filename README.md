# falcon9_reuse_first_stage
This is an applied capstone project from IBM data science course where the main porpose is to predict whether, in a specefic falcon 9 launch, the first stage could be reused, or not, giving us an estimate of the launch cost.

## Data collection using spaceX API
jupyter-labs-spacex-data-collection-apy file is the script used to collect data using spacex API.

## Data collection using web scraping
jupyter-labs-web-scraping file is the script used to collect data from a wikipedia web page.
